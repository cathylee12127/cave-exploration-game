# 题库数据说明

## 概述

本文档详细说明溶洞探秘互动小游戏的题库数据内容和结构。题库包含12道精心设计的溶洞科学知识题目，涵盖基础知识和进阶内容。

## 题库统计

- **总题目数**: 12道
- **基础题（10分）**: 6道
- **提升题（20分）**: 6道
- **每题选项数**: 3个（1个正确答案 + 2个干扰项）

## 题目列表

### 基础题（10分）

#### 1. 钟乳石是如何形成的？
- **正确答案**: 地下水中的碳酸钙沉积
- **干扰项**:
  - 岩浆冷却凝固
  - 风化作用形成
- **知识点**: 钟乳石的形成原理

#### 2. 石笋通常生长在溶洞的什么位置？
- **正确答案**: 溶洞底部
- **干扰项**:
  - 溶洞顶部
  - 溶洞侧壁
- **知识点**: 石笋的生长位置

#### 3. 石柱是由什么形成的？
- **正确答案**: 钟乳石和石笋连接
- **干扰项**:
  - 地震挤压
  - 人工雕刻
- **知识点**: 石柱的形成过程

#### 4. 溶洞主要是由什么岩石溶蚀形成的？
- **正确答案**: 石灰岩
- **干扰项**:
  - 花岗岩
  - 玄武岩
- **知识点**: 溶洞的岩石类型

#### 5. 钟乳石的生长速度大约是多少？
- **正确答案**: 每百年几厘米
- **干扰项**:
  - 每年1厘米
  - 每月1毫米
- **知识点**: 钟乳石的生长速度

#### 6. 溶洞中常见的水滴声是由什么引起的？
- **正确答案**: 地下水从钟乳石滴落
- **干扰项**:
  - 地下河流动
  - 岩石碰撞
- **知识点**: 溶洞中的声音来源

### 提升题（20分）

#### 7. 碳酸钙沉积形成钟乳石的化学反应过程中，起关键作用的是什么？
- **正确答案**: 二氧化碳的溶解和释放
- **干扰项**:
  - 氧气的氧化作用
  - 氮气的固定作用
- **知识点**: 钟乳石形成的化学原理

#### 8. 溶洞的形成需要经历多长时间？
- **正确答案**: 数万至数百万年
- **干扰项**:
  - 几十年
  - 几百年
- **知识点**: 溶洞的形成时间尺度

#### 9. 为什么溶洞内的温度相对恒定？
- **正确答案**: 深埋地下，不受外界气温影响
- **干扰项**:
  - 地热持续加温
  - 水流调节温度
- **知识点**: 溶洞的温度特征

#### 10. 溶洞中的"石花"是如何形成的？
- **正确答案**: 毛细作用和结晶作用
- **干扰项**:
  - 水流冲刷形成
  - 微生物堆积
- **知识点**: 石花的形成机制

#### 11. 中国最大的溶洞系统位于哪个省份？
- **正确答案**: 贵州省
- **干扰项**:
  - 云南省
  - 四川省
- **知识点**: 中国溶洞地理分布

#### 12. 溶洞中的"石幔"景观是如何形成的？
- **正确答案**: 水沿洞壁流淌时碳酸钙沉积
- **干扰项**:
  - 岩石层层剥落
  - 地下水位变化造成
- **知识点**: 石幔的形成过程

## 知识点覆盖

题库涵盖以下溶洞科学知识领域：

1. **地质形成**
   - 溶洞的岩石类型（石灰岩）
   - 溶洞的形成时间（数万至数百万年）

2. **钟乳石、石笋、石柱**
   - 钟乳石的形成原理（碳酸钙沉积）
   - 石笋的生长位置（溶洞底部）
   - 石柱的形成过程（钟乳石和石笋连接）
   - 钟乳石的生长速度（每百年几厘米）

3. **化学原理**
   - 碳酸钙沉积的化学反应（二氧化碳的溶解和释放）

4. **特殊景观**
   - 石花的形成（毛细作用和结晶作用）
   - 石幔的形成（水沿洞壁流淌时碳酸钙沉积）

5. **物理特征**
   - 溶洞温度恒定的原因（深埋地下）
   - 水滴声的来源（地下水从钟乳石滴落）

6. **地理知识**
   - 中国最大的溶洞系统（贵州省）

## 难度设计

### 基础题（10分）
- 面向大众认知水平
- 考察基本概念和常识
- 答案相对直观
- 适合初次接触溶洞知识的玩家

### 提升题（20分）
- 需要一定的科学知识背景
- 考察深层原理和机制
- 涉及化学、物理、地理等跨学科知识
- 适合对溶洞有一定了解的玩家

## 干扰项设计原则

1. **合理性**: 干扰项在表面上看起来合理，但实际上不正确
2. **区分度**: 干扰项与正确答案有明显区别，避免歧义
3. **教育性**: 通过排除干扰项，玩家可以学习到更多知识
4. **多样性**: 干扰项涵盖不同的错误类型（概念混淆、数值错误、因果倒置等）

## 使用方法

### 1. 初始化数据库表结构

```bash
npm run db:init
```

### 2. 插入题库数据

```bash
npm run db:seed
```

### 3. 验证题库数据（不需要数据库）

```bash
npm run db:verify
```

### 4. 运行测试

```bash
npm test database/seed.test.js
```

## 扩展题库

如果需要添加更多题目，请遵循以下步骤：

1. 在 `seed.js` 的 `questionsData` 数组中添加新题目
2. 确保新题目符合数据结构要求
3. 运行 `npm run db:verify` 验证数据结构
4. 运行 `npm test database/seed.test.js` 确保测试通过
5. 运行 `npm run db:seed` 重新插入数据
