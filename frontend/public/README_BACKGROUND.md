# 🖼️ 溶洞背景图片说明

## 📁 放置图片的位置

将你的溶洞背景图片放在这个文件夹（`frontend/public/`）中。

## 📝 文件命名

图片必须命名为以下之一：
- `cave-background.jpg`（推荐）
- `cave-background.png`

## 🎨 推荐图片规格

- **格式**：JPEG 或 PNG
- **分辨率**：1920x1080 或更高
- **文件大小**：500KB - 1MB
- **方向**：横向
- **内容**：溶洞内部，有钟乳石和石笋

## 🔍 如何获取图片

请查看项目根目录的 `如何添加溶洞背景图片.md` 文件，里面有详细的图片获取指南。

## ✨ 效果

添加图片后，游戏会：
- ✅ 使用真实溶洞照片作为背景
- ✅ 保留动态光影效果
- ✅ 保留水滴动画
- ✅ 保留交互点动画

## 🔄 如果没有图片

如果这个文件夹中没有 `cave-background.jpg` 或 `cave-background.png`，游戏会自动使用 Canvas 绘制的背景（当前的效果）。

## 📞 需要帮助？

如果遇到问题，请查看 `如何添加溶洞背景图片.md` 文件中的"常见问题"部分。
